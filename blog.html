✏️ How to publish a new post

Open data/blog-posts.json and add a new object to the array. The content field accepts HTML — use <p> for paragraphs, <h2> for headings, <blockquote> for quotes, <ul> for lists.

{
  "id": 4,
  "title": "My New Article Title",
  "category": "Teaching Reflections",
  "date": "2025-05-01",
  "readTime": "5 min read",
  "excerpt": "A short 1-2 sentence teaser shown on the card.",
  "content": "<p>Your full article goes here. Use HTML tags.</p><h2>Section heading</h2><p>More content.</p>"
}

Tip: make sure each post has a unique id number. Increment from the last one.
