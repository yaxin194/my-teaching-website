➕ How to add more vocabulary

Open the file data/vocabulary.json in any text editor. Copy an existing entry and fill in your own word, sentence, definition, and notes. Each entry follows this structure:

{
  "word": "your word",
  "phonetic": "/fəʊˈnetɪk/",
  "partOfSpeech": "noun / verb / adjective / etc.",
  "definition": "clear definition in English",
  "sentence": "A real sentence using the word naturally.",
  "sentenceCN": "中文翻译",
  "usageNote": "Tip about when and how to use this word.",
  "level": "B2",          ← A2 / B1 / B2 / C1 / C2
  "category": "Academic"  ← Academic / IELTS / ELT
}

Save the file and refresh the page — your new card will appear automatically.
