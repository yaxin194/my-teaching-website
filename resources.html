➕ How to add more resources

Open data/resources.json and add a new entry. To add a downloadable file, put the file in a files/ folder and set "link" to its path (e.g. "files/essay-template.pdf").

{
  "title": "Resource Title",
  "description": "What it contains and who it's for.",
  "category": "writing",  ← writing / speaking / vocab / grammar
  "level": "B2–C1",
  "tags": ["IELTS", "Essay"],
  "link": "files/my-resource.pdf",
  "linkText": "Download PDF"
}
